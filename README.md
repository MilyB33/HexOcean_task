## How to setup

### Production

    1. Clone the repository using `git clone https://github.com/MilyB33/HexOcean_task.git`
    2. Run `npm install`
    3. Run `npm run build`
    4. Run `npm run start`
    5. Server should run on 'http://localhost:4173/'

### Development

    1. Clone the repository using `git clone https://github.com/MilyB33/HexOcean_task.git`
    2. Run `npm install`
    3. Run `npm run dev`
    4. Server should run on 'http://localhost:3000/'
